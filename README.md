# CorrectlyAI - Form Auto-Fill Extension

A Chrome extension that automatically fills Google Forms using Google's Gemini AI and OpenAI's ChatGPT API.

## Setup Instructions

1. **Load the Extension:**
   - Open Chrome and go to `chrome://extensions/`
   - Enable "Developer mode" (toggle in top right)
   - Click "Load unpacked"
   - Select this folder

2. **Get Gemini API Key:**
   - Go to https://aistudio.google.com/app/apikey
   - Create a new API key
   - Copy the API key

3. **Configure Extension:**
   - Click the extension icon in Chrome toolbar
   - Paste your Gemini API key
   - Click "Save API Key"

4. **Use the Extension:**
   - Open any Google Form
   - Click the "AI Solve" button that appears on the form
   - The extension will automatically fill all questions using Gemini AI

## Features

- Reads all questions from Google Forms
- Includes options in prompts for accurate matching
- Automatically fills single choice, multiple choice, dropdowns, and text fields
- Shows answers on form (toggleable)

## Requirements

- Chrome browser
- Valid Gemini API key from Google AI Studio
